== Description ==

Need content? The Entertainment News plugin will display unique articles for your site.

This plugin comes with 2 shortcodes that can be used to add content to posts and pages.
It also comes with a widget to display news articles in a side column or somewhere in your theme.

The content is provided by Gloom Wire and is free to use providing you link back.
The plugin handles everything for you and has many options to choose from.

All html comes preset with css waiting to be customized.
If your technical, feel free to modify the css file to make this plugin look even better.

== Installation ==

Requires PHP version 5.3 or grater.

1. To install search for Entertainment News on your site's Add Plugins page.
To manually install, upload the Entertainment News plugin folder or zip file to your '/wp-content/plugins' folder.
If you need help, general instructions can be found at https://codex.wordpress.org/Managing_Plugins

2. Activate the plugin through the 'Plugins' menu in WordPress

3. Add the Entertainment News widget from your admin widgets page or post a shortcode below to a post/page.
[entertainment-news-summery]
[entertainment-news-articles]

Settings for the widget are on the widget page. The settings for the shortcodes are on the Entertainment News plugin page.

If you site is visted by millions of people a month, consider turning on cache to increase performance with this plugin.
There's a great tutorial here, http://dougal.gunters.org/blog/2006/07/21/using-the-wordpress-object-cache/
